Setup Guide

Get Repollect running locally in minutes.

Prerequisites

  • Python 3.11+
  • Node.js 18+
  • Ollama (for local LLM + embeddings) — or a cloud LLM provider

Backend Setup

1

Clone the repository

git clone https://github.com/muskiteer/cognee.git
cd cognee
2

Create virtual environment & install dependencies

cd app
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
3

Configure environment

cp .env.example .env
# Open .env and add your Groq, GitHub, Notion, and Discord tokens
4

Run the server

uvicorn main:app --host 0.0.0.0 --port 8000 --reload
# → API running at http://localhost:8000

Frontend Setup

1

Install dependencies

cd ../ui
npm install
2

Start the dev server

npm run dev
# → React UI running at http://localhost:5173

Environment Variables

Create a .env file in the app/ directory. Below is the full reference of supported variables.

Variable Required Description
LLM_PROVIDER Yes "ollama" or "gemini"
LLM_MODEL Yes e.g. "qwen3:4b", "gemini/gemini-2.5-flash"
LLM_ENDPOINT Ollama "http://localhost:11434/v1"
LLM_API_KEY Yes API key or "ollama" for local
EMBEDDING_PROVIDER Yes "ollama"
EMBEDDING_MODEL Yes "nomic-embed-text:latest"
EMBEDDING_ENDPOINT Yes "http://localhost:11434/api/embed"
EMBEDDING_DIMENSIONS Yes "768"
GROQ_API Chat Groq API key for LLM inference
GITHUB_PAT_TOKEN GitHub GitHub Personal Access Token
NOTION_TOKEN Notion Notion Integration Token (secret_...)
DISCORD_BOT_TOKEN Discord Discord Bot Token (MTAx...)
AUTO_REVIEW No True/False — enable auto-review scheduler
REVIEW_TIME_MIN No Interval in minutes for the review scheduler (default: 30)

LLM Provider Configuration

Repollect supports Ollama (local) and Gemini (cloud) as LLM providers. For detailed configuration options and supported models, see the Cognee LLM Providers documentation.

Embedding Provider Configuration

Embeddings can be configured via Ollama or FastEmbed. For detailed setup, see the Cognee Embedding Providers documentation.

Tip: For local development, Ollama is the fastest path. Install Ollama, pull qwen3:4b and nomic-embed-text, then set the environment variables as shown above.

Getting API Keys

Groq API Key

Used for fast LLM inference in the chat interface.

  1. Go to Groq Console
  2. Sign in and navigate to API Keys
  3. Click Create API Key, copy it, and paste it as GROQ_API

GitHub PAT

Required to fetch repositories, read issues/PRs, and post comments.

  1. Go to GitHub Settings > Developer Settings > Personal access tokens
  2. Click Generate new token (classic)
  3. Give it a name and select the repo scope
  4. Generate the token and paste it as GITHUB_PAT_TOKEN

Notion Integration Token

Required to search and read/write Notion pages.

  1. Go to Notion My Integrations
  2. Click New integration
  3. Name it "Repollect", select the associated workspace, and save
  4. Copy the Internal Integration Secret and paste it as NOTION_TOKEN
Critical: You must manually share the Notion pages/databases you want Repollect to access. Open a page in Notion, click the ... menu, go to Add connections, and select your "Repollect" integration.

Discord Bot Token

Required to read channel messages for the knowledge graph.

  1. Go to the Discord Developer Portal
  2. Click New Application and name it "Repollect"
  3. Go to the Bot tab
  4. Under Privileged Gateway Intents, enable Message Content Intent
  5. Click Reset Token, copy it, and paste as DISCORD_BOT_TOKEN
  6. Go to OAuth2 > URL Generator, check bot scope, and under permissions check Read Messages/View Channels and Read Message History
  7. Use the generated URL to invite the bot to your server
