Slash Commands

Ask anything. Do anything.

Specialized commands that combine live GitHub API data with knowledge graph context.

🐛
/issue <title>

Create a GitHub Issue directly from chat without leaving RepoCollect.

📋
/notion <title>

Create a Notion page from chat context and save it to your workspace.

🔍
iss{N}

Fetch any issue live from GitHub and ask questions grounded in its full context.

🔀
pr{N}

Pull any PR with review comments and discussion, then ask questions about it.

📊
diff{N}

Get an AI-powered explanation of a PR's code changes across up to 10 files.

👥
/contributors

List all contributors with commit counts — no question needed.

👤
@username

Get a full activity summary for any contributor, grounded in their actual PRs and issues.

🤖
Auto-Review

Background scheduler posts AI-generated reviews on new Issues & PRs using graph history. Set AUTO_REVIEW=True.
