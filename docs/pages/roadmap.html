Development Roadmap

The project was built over 7 days for the WeMakeDevs × Cognee Hackathon (Jun 29 – Jul 5, 2026):

DayGoal
Day 1 (Jun 29)Set up Cognee, build GitHub ingestion pipeline
Day 2 (Jun 30)Build Smart Memory Router (tagging + filtering)
Day 3 (Jul 1)Discord ingestion + Notion ingestion
Day 4 (Jul 2)Build Cognee memory tools (remember, recall, forget, improve)
Day 5 (Jul 3)Web UI / chat interface + REST API refinement
Day 6 (Jul 4)Polish, test on real OSS repos, fix edge cases
Day 7 (Jul 5)Record demo video, finalize README, submit

Phase-Based Feature Delivery

The project was built in 5 phases, each building on the previous one:

PhaseDeliverable
Part 1Projects, authentication, dataset (owner/repo) setup
Part 2Initial ingestion, incremental sync, background jobs
Part 3WebSocket logs, scheduler, automatic PR reviews & issue monitoring
Part 4Multi-chat interface with SQLite conversation history
Part 5AI actions (Open PR, Write to Notion, future GitHub/Discord actions)
Vision: Repollect evolves from a passive knowledge base into an active, intelligent contributor — catching regressions, identifying knowledge gaps, and helping maintainers spend less time answering the same questions.
